# Personal-Portfolio-Website-
Developed a full-stack personal portfolio website to showcase projects, skills, and contact information. The platform integrates a responsive React frontend with a Node.js/Express backend and MongoDB database, allowing dynamic storage and retrieval of project data.
📌 Personal Portfolio Website (Full Stack)
A full-stack personal portfolio web application built using Node.js, Express, MongoDB, and MySQL.
It showcases projects dynamically and includes a contact system.
🚀 Features
🧑‍💻 Dynamic Projects Section (MongoDB)
📩 Contact Form (MySQL)
🔗 REST API (Express.js)
⚡ Dual Database System (MongoDB + MySQL)
🌐 Ready for deployment (Render / Vercel)
🔐 Scalable backend architecture
